# 🎒 سامانه پوشه کار تعاملی

یک سیستم جامع برای مدیریت پوشه کار، هدف‌گذاری، خودارزیابی و پاداش دانش‌آموزان.

## ✨ ویژگی‌ها

- **پورتال دانش‌آموز**: پوشه کار، هدف‌گذاری SMART، خودارزیابی ماهانه
- **پورتال معلم**: مدیریت دانش‌آموزان، پاداش‌ها، کتابخانه
- **پورتال والدین**: مشاهده پیشرفت، امتیازات و بازخورد
- **پورتال مدیر**: مدیریت کامل مدرسه و معاونین
- **سیستم پاداش**: امتیازات، نشان‌ها و ویترین جوایز
- **گزارش‌گیری پیشرفته**: خروجی PDF و Word
- **امنیت**: رمزنگاری، پشتیبان‌گیری و لاگینگ

## 🚀 نصب و راه‌اندازی

### نیازمندی‌ها
- یک مرورگر مدرن (Chrome, Firefox, Safari, Edge)
- سرور محلی (Python, Node.js یا Live Server)
